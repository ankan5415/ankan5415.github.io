OpenFlite - A Developer Drone.

About

Recently, I started working on a Developer drone. I have a deep passion for building and flying drones, especially using other technologies such as 3D printing. Because I've recently been really intersted in AI and it's applications, I wanted to use this tech to build a drone that can perform automated tasks and use IoT systems to create easy data analysis and data transfer. I am currently building drone testing rigs, creating drone ML agents on unity and preparing an effective communication system between the drone and the computer. Check out my article to see a more in-depth view of the project and where it came from here.
